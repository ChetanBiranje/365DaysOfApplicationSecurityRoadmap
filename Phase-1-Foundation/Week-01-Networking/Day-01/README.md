### What is Application Security?

Application Security (AppSec) involves finding and fixing security vulnerabilities in software applications before hackers can exploit them.

**Key Responsibilities:**
- Finding vulnerabilities in web/mobile applications
- Reviewing code for security flaws
- Testing APIs for weaknesses
- Working with developers to fix issues
- Building secure applications

**Why AppSec?**
- High demand (shortage of skilled professionals)
- Great salary (â‚¹7-30 LPA in India)
- Challenging and never boring
- Work from anywhere
- Continuous learning
